## Aliases and grouping

Use the `@include-docs` annotation to

- specify aliases for the function and constants that can be used as `of` identifiers for the `ArgTypes` and/or
- group multiple exports into a single `ArgTypes` table.

```typescript
/**
 * Current environment
 *
 * @include-docs Env, LibraryInfo, EnvironmentInfo
 */
export const ENVIRONMENT = 'production';

/**
 * Returns the current environment
 *
 * @include-docs LibraryInfo, EnvironmentInfo
 */
export function getEnvironment(): string {
	// ...
}
```
