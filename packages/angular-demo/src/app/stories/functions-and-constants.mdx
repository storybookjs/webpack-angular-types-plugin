## @group-docs

Group multiple exports into a single ArgTypes using the `@group-docs` annotation.

```typescript
/**
 * Returns the current environment
 *
 * @include-docs
 * @group-docs LibraryInfo, EnvironmentInfo
 */
export function getEnvironment(): string {
	// ...
}
```
